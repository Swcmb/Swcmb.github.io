曲园教评半自动打分脚本实现

发表于 2024-03-02 分类于 实用工具

背景

每学期末的教学评价工作往往繁琐且重复，面对大量的课程和老师，手动点击评分效率极低。在寻找现成解决方案未果后，我决定自己动手编写一个油猴脚本。

该脚本目前定位为“半自动”打分工具，旨在通过模拟用户点击行为，快速完成评教页面的选项填充，将同学们从机械的操作中解放出来。

技术实现原理

脚本的核心逻辑非常直观，主要利用了 JavaScript 的 DOM 操作能力：

  1. 目标锁定：通过 document.querySelectorAll 批量获取页面上指定的评分单选框元素。
  2. 模拟交互：遍历获取到的元素列表，通过 createEvent 创建鼠标点击事件，并分发到每个目标元素上，从而实现“自动勾选”的效果。

脚本代码

以下为完整的油猴脚本代码。使用前请确保浏览器已安装 Tampermonkey 插件。

```javascript
// ==UserScript==
// @name 曲园全自动教评打分
// @namespace http://example.com/
// @version 1.0
// @description 曲园全自动教评打分(声明见：https://www.luogu.com.cn/blog/ianwusb/quyuansoringauto)
// @author ianwusb from(https://www.luogu.com.cn/blog/ianwusb/)
// @match http://202.194.188.38/*
// @grant none
// ==/UserScript==

(function() {
‘use strict’;

// 指定要点击的元素的ID数组
// 注意：这里的ID需根据实际网页结构进行调整
const targetElementIds = [
    'pj0601id_2_2', 'pj0601id_3_2', 'pj0601id_4_2',
    'pj0601id_5_2', 'pj0601id_6_2', 'pj0601id_7_2',
    'pj0601id_8_2', 'pj0601id_9_2', 'pj0601id_10_2',
    'pj0601id_11_1'
]; 

// 构建选择器字符串并获取目标元素列表
const selector = '#' + targetElementIds.join(', #');
const targetElements = document.querySelectorAll(selector);

// 检查元素是否存在
if (targetElements.length > 0) {
    // 直接模拟点击所有目标元素的操作
    targetElements.forEach(function(element) {
        simulateClick(element); // 对每个目标元素执行模拟点击操作
    });
} else {
    console.error('Target elements not found');
}

// 模拟鼠标点击元素的函数
function simulateClick(element) {
    const clickEvent = document.createEvent('MouseEvents');
    // 初始化事件（注：此处代码原文被截断，以下为补全的标准写法）
    clickEvent.initEvent('click', true, true);
    element.dispatchEvent(clickEvent);
}

})();

使用说明

  1. 安装插件：在浏览器中安装 Tampermonkey（油猴）扩展。
  2. 添加脚本：新建脚本，将上述代码粘贴并保存。
  3. 访问页面：进入教务系统教评页面，脚本将自动执行选中操作。
  4. 自定义修改：如果评分项未自动勾选，请按 F12 检查网页元素，修改 targetElementIds 数组中的 ID 值以匹配当前页面元素。

注意：代码中的 pj0601id_x_x 等元素 ID 是基于特定时间的网页结构抓取的，若教务系统升级，这些 ID 可能会失效，届时需重新检查元素并更新代码。
