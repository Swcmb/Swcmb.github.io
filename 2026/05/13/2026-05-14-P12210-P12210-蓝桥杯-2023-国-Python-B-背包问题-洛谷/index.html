P12210 - P12210 [蓝桥杯 2023 国 Python B] 背包问题 - 洛谷

算法名称：背包问题

1. 标题

[洛谷 P12210] 背包问题 - 核心优化技巧

2. 题目概览

  • 题目来源: https://www.luogu.com.cn/problem/P12210
  • 题目难度: ★★★☆☆
  • 标签分类: 算法, 数据结构
  • 核心考点: 动态规划、贪心算法

3. 题意解析

  • 问题描述: 给定三个整数 pa1, pa2, pa3，以及两个整数 cntA 和 cntB，求在不超过背包容量的前提下，如何分配这些物品，使得背包中的物品数量最大化。
  • 输入格式: 输入为一个整数数组，包含三个元素 pa1, pa2, pa3，以及两个整数 cntA 和 cntB。
  • 输出格式: 返回一个整数，表示最大物品数量。

4. 算法思路（核心部分）

4.1 问题分析

  • 问题本质: 这是一个经典的背包问题，需要找到最优解，即在不超过背包容量的前提下，最大化背包中的物品数量。
  • 关键观察点: 每个物品的重量和价值，背包的容量限制。
  • 数据范围: 物品重量和价值的范围是已知的，背包容量是固定的。

4.2 算法原理

  • 朴素思路: 遍历所有可能的组合，计算每种组合的物品数量，然后选择最大的那个。这种方法的时间复杂度是 O(n^3)。
  • 优化路径: 使用动态规划来避免重复计算。首先创建一个二维数组 dp，其中 dp[i][j] 表示前 i 个物品中选取 j 个物品的最大价值。然后，通过状态转移方程更新这个数组。最后，从 dp 数组中选择最大值作为结果。这种方法的时间复杂度是 O(n^2 * m)，其中 n 是物品数量，m 是物品种类数。
  • 算法选择: 由于物品数量较多，且背包容量有限，使用动态规划可以显著提高算法效率。
  • 核心思想: 通过构建一个二维数组来存储子问题的解，避免了重复计算，从而将时间复杂度降低到 O(n^2 * m)。

4.3 数学推导

  • 关键公式:
    • 状态转移方程：dp[i][j] = max(dp[i-1][k] + value[j], dp[i-1][j])
    • 最终结果：ans = max(dp[cntA][cntB])
  • 正确性证明: 根据动态规划的性质，如果当前状态的值大于或等于其前一个状态的值，则当前状态的值就是前一个状态的值加当前物品的价值；否则，当前状态的值就是前一个状态的值加当前物品的价值。因此，ans = max(dp[cntA][cntB]) 就是正确的结果。
  • 复杂度分析: 时间复杂度为 O(n^2 * m)，空间复杂度为 O(n^2)。

4.4 多种解法对比

解法 时间复杂度 空间复杂度 适用场景 代码难度
解法1 O(n^3) O(n^2) 小规模数据 简单
解法2 O(n^2 * m) O(n^2) 中等规模 中等

5.2 逐行解析

  • pa1, pa2, pa3, vA, vB = map(int, input().split()): 读取输入，获取物品重量和价值，以及背包容量。
  • cntA, cntB = [int(x) for x in input().split()]: 读取剩余物品的数量和总价值。
  • vB = min(vA, vB): 初始化背包容量为最小值。
  • ans = float('-inf'): 初始化最大物品数量为负无穷大。
  • for i in range(min(cntA, cntB)):: 遍历所有可能的组合。
  • vB -= pa1: 如果当前物品重量小于等于背包容量，则从背包中取出该物品。
  • if vB < 0:: 如果背包容量不足，则跳出循环。
  • ans = max(ans, (vB, pa1)): 如果当前物品重量大于等于背包容量，则更新最大物品数量。
  • for j in range(i+1, min(cntA, cntB)-1):: 继续遍历其他物品组合。
  • vB -= pa2: 如果当前物品重量小于等于背包容量，则从背包中取出该物品。
  • if vB < 0:: 如果背包容量不足，则跳出循环。
  • ans = max(ans, (vB, pa2)): 如果当前物品重量大于等于背包容量，则更新最大物品数量。
  • for k in range(i+1-j, min(cntA, cntB)-1):: 继续遍历其他物品组合。
  • vB -= pa3: 如果当前物品重量小于等于背包容量，则从背包中取出该物品。
  • if vB < 0:: 如果背包容量不足，则跳出循环。
  • ans = max(ans, (vB, pa3)): 如果当前物品重量大于等于背包容量，则更新最大物品数量。
  • return ans: 返回最大物品数量。

6. 复杂度分析

  • 时间复杂度: O(n^2 * m)，其中 n 是物品数量，m 是物品种类数。因为需要遍历所有可能的组合，并且对于每个组合都需要进行多次比较和操作。
  • 空间复杂度: O(n^2)，用于存储二维数组 dp。
