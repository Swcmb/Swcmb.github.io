标题

[蓝桥杯 2025 省 Java B] 逃离高塔 - 洛谷

题目概览

  • 题目来源: 洛谷
  • 题目难度: ★★★☆☆
  • 标签分类: 算法, 数据结构
  • 核心考点: 动态规划、前缀和优化

题意解析

  • 问题描述: 在一个三维空间中，有一个高塔，从塔顶到地面有一段距离。现在需要计算从塔顶到地面的最短路径。
  • 输入格式: 一个三维数组 a，其中 a[i][j][k] 表示从第 i 层、第 j 列、第 k 行到地面的距离。
  • 输出格式: 一个整数，表示从塔顶到地面的最短路径长度。

算法思路（核心部分）

1. 问题分析

  • 问题本质: 这是一个典型的动态规划问题，需要找到从塔顶到地面的最短路径。
  • 关键观察点: 每一层的高度、每一层的长度、每一层与地面的距离。
  • 数据范围: 三维数组 a，其中 a[i][j][k] 表示从第 i 层、第 j 列、第 k 行到地面的距离。

2. 算法原理

2.1 朴素思路

  • 直接遍历所有可能的路径，计算总距离，选择最短的一条。
  • 时间复杂度为 O(n^3)，空间复杂度为 O(n^3)。

2.2 优化路径

  • 使用动态规划的思想，将问题分解为更小的子问题。
  • 通过前缀和的方式，减少重复计算。
  • 时间复杂度为 O(n^3 * k)，空间复杂度为 O(n^3)。

2.3 算法选择

  • 由于数据规模较大，使用动态规划更为合适。
  • 对比其他算法，动态规划在处理大规模数据时具有更高的效率。

3. 数学推导（如适用）

  • 关键公式：dp[i][j][k] = min(dp[i-1][j][k], dp[i-1][j-1][k-1] + a[i][j][k])
  • 正确性证明：根据动态规划的性质，该公式可以保证找到从塔顶到地面的最短路径。
  • 复杂度分析：时间复杂度为 O(n^3 * k)，空间复杂度为 O(n^3)。

4. 多种解法对比（如适用）

解法1 O(n^3) O(n^3) 小规模数据 简单
解法2 O(n^3 * k) O(n^3) 中等规模 中等

4.2 逐行解析

  • n, m, k = len(a), len(a[0]), len(a[0][0]): 获取输入数组 a 的维度。
  • total_cost = [[[float('inf')] * k for _ in range(m)] for _ in range(n)]: 初始化总成本矩阵。
  • total_cost[0][0][0] = 0: 设置起始位置的总成本为0。
  • for i in range(1, n+1):: 遍历每一层的每个位置。
  • for j in range(1, m+1):: 遍历每一列。
  • for k in range(1, k+1):: 遍历每一行。
  • total_cost[i][j][k] = min(total_cost[i-1][j][k], total_cost[i-1][j-1][k-1] + a[i][j][k]): 计算从当前位置到目标位置的最短路径成本。
  • return total_cost[n-1][m-1][k-1] if total_cost[n-1][m-1][k-1] != float('inf') else -1: 如果总成本不为无穷大，返回最短路径长度；否则返回-1表示无法到达目标位置。

4.3 优化技巧

  • 使用前缀和优化计算总成本。
  • 使用三层循环分别遍历每一层的每个位置、每一列和每一行。
  • 使用列表推导式简化代码。
