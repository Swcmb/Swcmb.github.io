[洛谷 P12843] 生日相遇问题详解

1. 标题

洛谷 P12843 生日相遇问题详解

2. 题意简述

给定一个日期范围，计算在这个范围内，某两个特定日期落在同一星期的年份数量。

输入内容：月份 m, 日 d1, 月份 n, 日 d2, 年份范围 k（从2025年开始计算）。
输出要求：输出所有满足条件的年份，如果没有满足条件的年份，则输出”No Answer”。
数据范围：m, d1, n, d2 为 1 到 12，1 到 31 的整数，k 为 0 到 1000 的整数。

3. 思路分析

  • 朴素解法：遍历指定年份范围内的每一年，对于每一年的两个日期，使用基姆拉尔森计算公式计算星期几，然后比较两个日期是否落在同一星期。这种方法的时间复杂度为 $O(n \times k)$，其中 n 为年份范围内的年数，k 为年份范围内的年数，显然效率低下。
  • 优化方向：由于年份范围固定，可以考虑使用数学方法来减少比较次数。例如，我们可以计算两个日期相对于2025年的星期数，然后比较这两个星期数是否相同。
  • 最终解法：使用基姆拉尔森计算公式计算两个日期相对于2025年的星期数，然后比较这两个星期数是否相同。如果相同，则记录下这一年。这种方法的时间复杂度为 $O(k)$，空间复杂度为 $O(1)$。

4. 复杂度分析

  • 时间复杂度：$O(k)$，其中 k 为年份范围内的年数。
  • 空间复杂度：$O(1)$。

5. 核心代码 + 注释

```python
def isLeap(y):
return (y % 4 == 0 and y % 100 != 0) or (y % 400 == 0)

def adjust(y, m, d):
if m == 2 and d == 29:
return (2, 28) if not isLeap(y) else (2, 29)
return (m, d)

def week(y, m, d):
if m == 1 or m == 2:
y -= 1
m += 12
w = (d + 2m + 3(m+1)//5 + y + y//4 - y//100 + y//400 + 1) % 7
return w

输入

m, d1, n, d2, k = map(int, input().split())
res = []

for year in range(2025, 2025 + k):
m1, d1_adj = adjust(year, m, d1)
n1, d2_adj = adjust(year, n, d2)
if week(year, m1, d1_adj) == week(year, n1, d2_adj):
res.append(year)

if res:
print(“\n”.join(map(str, res)))
else:
print(“No Answer”)

7. 易错点 & 收获

  • 易错点：处理闰年时，需要正确地调整2月29日为2月28日。
  • 收获：本题教会了如何使用基姆拉尔森计算公式来计算星期几，以及如何通过数学方法优化算法。
