[洛谷 P12278] 设置密码 - 洛谷

题意简述

在本次洛谷 P12278 的题目中，我们需要编写一个程序来判断用户设置的密码强度。密码由大写字母、小写字母、数字和特殊字符组成。根据密码的不同组合情况，我们将其分为强密码、中密码和弱密码。具体规则如下：

  • 强密码：长度至少为12，且必须包含大写字母、小写字母、数字和至少一种特殊字符。
  • 中密码：长度至少为8，且至少包含两种字符类型。
  • 弱密码：长度至少为6。

思路分析

我们将从暴力解法开始，逐步优化算法，最终得到高效的解决方案。

朴素解法

最直观的方法是遍历所有可能的密码组合，并检查每个密码是否符合上述规则。然而，这种方法的时间复杂度为 $O(n^4)$，在实际应用中会非常慢，甚至无法在合理时间内完成。

优化方向

我们可以通过减少不必要的检查来优化算法。例如，一旦确定密码不符合强密码的条件，就可以立即返回结果，而不需要继续检查其他条件。

最终解法

我们定义以下状态：

  • has_upper: 密码中是否包含大写字母。
  • has_lowercase: 密码中是否包含小写字母。
  • has_digits: 密码中是否包含数字。
  • has_special_chars: 密码中是否包含特殊字符。
  • spec_count: 特殊字符的种类数。

状态转移方程如下：

  • has_upper = any(c in uppercase for c in st)
  • has_lowercase = any(c in lowercase for c in st)
  • has_digits = any(c in digits for c in st)
  • has_special_chars = len(set([c for c in st if c in special_chars])) > 0
  • spec_count = len(has_special_chars)

根据这些状态，我们可以编写如下代码：

```python
def check_password_strength(st):
valid_chars = uppercase + lowercase + digits + special_chars
if not all(c in valid_chars for c in st):
return 0 # 不合法

has_up = has_uppercase(st)
has_low = has_lowercase(st)
has_dig = has_digits(st)
has_spec = has_special_chars(st)
spec_count = len(has_spec)

if len(st) >= 12:
    if has_up and has_low and has_dig and spec_count >= 1:
        return 3
    if spec_count >= 3 and count_char_types(st) >= 3:
        return 3

if len(st) >= 8:
    char_types = count_char_types(st)
    if char_types >= 2:
        return 2

if len(st) >= 6:
    return 1

return 0  # 不合法

复杂度分析

  • 时间复杂度：$O(n)$，其中 $n$ 是密码的长度。我们只需要遍历一次密码字符串。
  • 空间复杂度：$O(1)$，我们只使用了常数个额外变量。

易错点 & 收获

  • 边界处理：确保密码字符串不为空，否则会导致 any 函数抛出异常。
  • 特殊字符处理：注意特殊字符的计数逻辑，确保正确统计种类数。
