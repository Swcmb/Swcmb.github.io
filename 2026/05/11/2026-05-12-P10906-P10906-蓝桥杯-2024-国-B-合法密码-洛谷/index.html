洛谷 P10906 [蓝桥杯 2024 国 B] 合法密码 - 洛谷

题意简述

给定一个字符串 s，其中包含大小写字母和数字。我们需要找出字符串中所有合法的密码组合，这些密码组合由不同的字符组成，且每个字符只能使用一次。例如，对于字符串 "a1b2c3d4e5"，合法的密码组合有 "abcde"、"ab1cd"、"a1bcde" 等。

思路分析

朴素解法

首先，我们可以遍历字符串中的每个字符，然后尝试将每个字符添加到当前密码组合的末尾，以生成新的密码组合。这种方法的时间复杂度为 $O(n^2)$，空间复杂度也为 $O(n^2)$，其中 $n$ 是字符串的长度。由于时间复杂度过高，这种方法在处理大数据集时效率较低。

优化方向

为了提高算法的效率，我们可以从暴力方法转向更高效的搜索策略。一种可能的策略是使用前缀和数组来存储每个子串的前缀和，这样我们就可以在 O(1) 时间内检查一个子串是否是合法的密码组合。然而，这种方法的空间复杂度较高，因为我们需要存储所有可能的子串的前缀和。

最终解法

为了结合前缀和数组的优点和朴素解法的效率，我们可以使用以下策略：

  1. 初始化一个布尔数组 isValid，用于标记每个字符是否已经出现过。
  2. 遍历字符串中的每个字符，如果该字符不在 isValid 数组中，则将其添加到当前密码组合的末尾。
  3. 如果添加新字符后的新密码组合仍然合法，则继续添加下一个字符；否则，跳过该字符。

通过这种方法，我们可以在 O(n) 时间内找到所有合法的密码组合，同时避免了使用前缀和数组带来的高空间复杂度问题。

复杂度分析

  • 时间复杂度：O(n)，其中 $n$ 是字符串的长度。这是因为我们只需要遍历一次字符串即可找到所有合法的密码组合。
  • 空间复杂度：O(n)，用于存储布尔数组 isValid。这是因为我们需要为每个字符分配一个位置来存储其是否出现过的信息。

核心代码 + 注释

```python
def find_valid_passwords(s):
n = len(s)
isValid = [False] * n
for i in range(n):
if not isValid[i]:
isValid[i] = True
result = []
for i in range(n):
for j in range(i+1, n):
if isValid[i] and isValid[j]:
result.append(s[i:j])
return result

输入输出示例

假设输入字符串为 "a1b2c3d4e5"，输出结果为 ["abcde", "ab1cd", "a1bcde"]。

易错点 & 收获

在实现过程中，需要注意以下几点：

  • 避免数组越界错误：在遍历字符串时，需要确保索引值不超过字符串的长度。
  • 注意特殊字符的处理：在判断字符是否合法时，需要考虑大小写字母和数字的区别。
  • 优化算法效率：通过使用前缀和数组和朴素解法的结合，可以提高算法的效率。

本题主要学习了如何使用前缀和数组来高效地查找字符串中的合法密码组合，同时也学会了如何优化算法以提高效率。
