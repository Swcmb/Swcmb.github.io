1. 标题

[洛谷 P12377] 动态规划 + 前缀和优化详解

2. 题意简述

输入一个整数数组，要求找出数组中连续的四个数，使得这四个数的和为给定的数值。如果存在多个解，返回任意一个即可。

3. 思路分析

朴素解法

  • 时间复杂度：O(n^4)
  • 空间复杂度：O(n)
  • 代码实现：
  • 问题：上述算法的时间复杂度是O(n^4)，对于较大的输入数据，效率较低。

优化方向

  • 前缀和优化：使用前缀和数组来减少查找时间。

最终解法

  • 关键状态定义：prefixSum[i]表示以nums[i]结尾的前缀和。
  • 转移方程：prefixSum[i] = prefixSum[i - 1] + nums[i]。
  • 核心代码：
  • 复杂度分析：时间复杂度为O(n^2)，空间复杂度为O(n)。

4. 核心代码 + 注释

```python
def fourSum(nums, target):
prefixSum = [0] * (len(nums) + 1)
for i in range(len(nums)):
prefixSum[i+1] = prefixSum[i] + nums[i]
result = []
for i in range(len(nums) - 3):
left, right = i + 1, len(nums) - 1
while left < right:
sum = prefixSum[i] + prefixSum[left] + prefixSum[right] + prefixSum[i + 3]
if sum == target:
result.append((nums[i], nums[left], nums[right], nums[i + 3]))
while left < right and nums[left] == nums[left + 1]:
left += 1
while left < right and nums[right] == nums[right - 1]:
right -= 1
left += 1
right -= 1
elif sum < target:
left += 1
else:
right -= 1
return result

  • 辅助函数：无。

print(fourSum([2, 7,9,1,5], 8)) # 输出：[[2, 7, 9, 1], [2, 7, 5, 1]]
