[洛谷 P10905] 回文字符串 - 洛谷

题意简述

本题是关于判断回文字符串的问题。输入包含多组测试数据，每组数据首先给出一个整数 ( T )，表示接下来有 ( T ) 组字符串，然后每组字符串由小写字母组成。对于每组字符串，需要判断其是否可以通过删除一些字符（也可以不删除）变成回文字符串。输出结果为 “Yes” 或 “No”。

思路分析

朴素解法

最直观的方法是对于每个字符串，分别检查它是否是回文字符串。具体来说，从字符串的两端开始，逐个比较字符是否相同，直到中间相遇或者发现不同的字符为止。这种方法的时间复杂度为 ( O(n^2) )，其中 ( n ) 是字符串的长度。由于需要处理多组字符串，总的时间复杂度为 ( O(Tn^2) )，在字符串长度较长或测试数据较多时，这种方法会非常慢。

优化方向

我们可以通过预处理每个字符在字符串中出现的次数来优化这个过程。具体来说，我们可以使用一个二维数组 ( cnt )，其中 ( cnt[i][j] ) 表示字符 ( i ) 在字符串的第 ( j ) 个位置之前出现的次数。这样，对于每个字符串，我们只需要检查 ( cnt ) 数组来确定是否可以通过删除一些字符使其成为回文字符串。

最终解法

定义状态：( dp[i][j][k] ) 表示从第 ( i ) 个字符到第 ( j ) 个字符，且中间跳过了 ( k ) 个字符后，形成的子串是否是回文字符串。

转移方程：

  1. 如果 ( s[i] == s[j] )，则 ( dp[i][j][k] = dp[i+1][j-1][k] )（如果 ( k \geq 0 )）。
  2. 如果 ( s[i] != s[j] )，则 ( dp[i][j][k] = dp[i+1][j][k-1] ) 或 ( dp[i][j-1][k] )（如果 ( k > 0 )）。

初始化：( dp[i][i][0] = true ) 对于所有 ( i )，因为单个字符总是回文的。

最终结果：对于每个字符串，检查 ( dp[0][n-1][k] ) 是否为真，其中 ( n ) 是字符串的长度。

时间复杂度：( O(n^2) )，其中 ( n ) 是字符串的长度。

空间复杂度：( O(n^2) )，用于存储 ( cnt ) 数组。

复杂度分析

  • 时间复杂度：( O(n^2) )
  • 空间复杂度：( O(n^2) )

核心代码 + 注释

```python
import sys

def solve() -> None:
data = sys.stdin.read().split()
if not data:
return
T = int(data[0])
out = []
for in range(T):
s = data[1]
n = len(s)
cnt = [[0] * 26 for in range(n)]

    # 计算每个字符出现的次数
    for i in range(n):
        cnt[i][ord(s[i]) - ord('a')] += 1

    flag = True
    # 检查是否可以通过删除一些字符使其成为回文字符串
    for i in range(n // 2):
        if cnt[i][ord(s[i]) - ord('a')] > 0 and cnt[n - i - 1][ord(s[i]) - ord('a')] > 0:
            flag = False
            break

    out.append("Yes" if flag else "No")

sys.stdout.write("\n".join(out))

if name == “main“:
solve()

输入：

abc

aabbcc
输出：

Yes
No

易错点 & 收获

  • 注意处理空字符串的情况。
  • 在计算字符出现次数时，需要确保字符在 ( a ) 到 ( z ) 的范围内。
