[蓝桥杯 2024省 C] 回文字符串 - 算法与数据结构题解

题目概览

  • 题目来源: 洛谷
  • 题目难度: ★★★☆☆
  • 标签分类: 算法, 数据结构
  • 核心考点: 回文字符串的检测

题意解析

  • 问题描述: 给定一个字符串，判断它是否为回文字符串。
  • 输入格式: 字符串类型，长度不超过10^5。
  • 输出格式: 布尔值，True表示是回文字符串，False表示不是。
  • 样例分析: 例如输入”radar”，输出True。

算法思路（核心部分）

1. 问题分析

  • 回文字符串的定义：正读和反读都一样的字符串。
  • 核心挑战在于如何高效地判断一个字符串是否是回文。

2. 算法原理

  • 朴素思路: 遍历字符串，逐个比较字符。
  • 优化路径: 使用双指针技术，分别从两端向中间移动，比较是否相等。
  • 算法选择: 由于需要处理大字符串，使用双指针可以有效减少时间复杂度。
  • 核心思想: 利用双指针技巧，提高算法效率。

3. 数学推导（如适用）

  • 关键公式：s[i] != s[j] 当且仅当 i < j 且 s[i] != s[n - j - 1]。
  • 正确性证明：通过双指针验证。
  • 复杂度分析：O(n)。

4. 多种解法对比（如适用）

解法 时间复杂度 空间复杂度 适用场景 代码难度
解法1 O(n) O(1) 小规模数据 简单
解法2 O(n log n) O(n) 中等规模 中等

代码详解

核心代码段

```python
def is_palindrome(s: str) -> bool:
n = len(s)
left, right = 0, n - 1
while left < right:
if s[left] != s[right]:
return False
left += 1
right -= 1
return True

逐行解析

  • def is_palindrome(s: str) -> bool: 定义函数，参数为字符串，返回值为布尔值。
  • n = len(s) 获取字符串长度。
  • left, right = 0, n - 1 初始化左右指针。
  • while left < right: 循环条件，直到左指针不小于右指针。
  • if s[left] != s[right]: 如果左右指针指向的字符不相等，则返回False。
  • left += 1 将左指针向右移动一位。
  • right -= 1 将右指针向左移动一位。
  • return True 如果所有字符都相等，则返回True。

优化技巧

  • 使用原地修改方法，避免额外的存储空间。
  • 在循环中直接比较字符，无需额外空间。

复杂度分析

  • 时间复杂度: O(n)。
  • 空间复杂度: O(1)。
  • 复杂度对比: 相比其他解法，本算法具有更低的时间和空间复杂度。

is_palindrome(“radar”) # True
