VBS脚本实战：实现QQ消息自动发送（早自习请假自动化）

背景：痛点与解决方案

对于许多需要早起参加早自习的同学来说，凌晨6点前的请假流程往往意味着打断睡眠，甚至导致整夜休息不佳。为了解决这一痛点，我们可以利用 Windows 系统原生的 VBS（VBScript）脚本结合任务计划程序，实现 QQ 消息的定时自动发送。

本教程将指导你如何配置自动化脚本，让你在休息时间完成请假手续，优化时间管理。

一、环境准备与配置

在开始之前，我们需要准备脚本文件并确保运行环境正确。

1. 获取脚本文件

首先下载核心脚本文件：

下载链接：abcd.vbs

2. 运行环境修复（如需）

VBS 脚本通常由系统自带的 wscript.exe 解析运行。如果运行脚本时出现报错，可能是系统环境缺失或关联错误。请尝试以下修复步骤：

  1. 下载备用解析器：wscript.exe。
  2. 将下载的 wscript.exe 文件放置在系统盘根目录下（即 C:\ 目录）。

3. 脚本参数配置

右键点击下载好的 abcd.vbs 文件，选择“编辑”或使用记事本打开。

找到代码中的字符串占位符，将其修改为你需要发送的请假内容。
关键提示：

  • 请将“你要说的话”替换为具体的请假事由。
  • 切勿在文本中换行，否则会导致脚本语法错误。

修改完成后保存并关闭文件。

二、配置 Windows 任务计划程序

为了让脚本在指定时间自动运行，我们需要利用 Windows 的“任务计划程序”。

步骤 1：打开任务计划程序

按下 Win + S 组合键，在搜索框输入“任务计划程序”并打开。

步骤 2：创建基本任务

按照以下图示流程进行操作：



步骤 3：设置触发时间

在时间设置界面，请务必注意一个关键技巧：

  • 选择一个过去的时间点。
  • 这是为了确保任务处于“就绪”状态，配合特定的重复策略（具体逻辑视脚本设计而定），防止系统时间同步问题导致任务未执行。



步骤 4：指定脚本路径

在“操作”步骤中，选择“启动程序”，然后浏览并选中你存放 abcd.vbs 的路径。




完成设置后退出即可。

三、系统电源管理设置（关键）

为了保证脚本在凌晨准时执行，电脑必须保持开机状态且不能进入休眠。

  1. 打开“此电脑”或“文件资源管理器”，在上方地址栏输入 控制面板 并回车。

  2. 找到并点击“电源选项”。

  3. 将计算机睡眠设置更改为“从不”，确保整晚通电且运行。

四、使用指南

准备工作完成后，实际操作非常简单：

  1. 打开目标窗口：睡觉前，打开 QQ 中负责请假或纪检的专用聊天窗口。
  2. 保持焦点：确保光标停留在消息输入框内（即聊天窗口处于前台激活状态），不要最小化。
  3. 供电保障：确保笔记本电脑接通电源，电量充足。
  4. 等待执行：一切准备就绪后，你就可以安心休息了，脚本将在预定时间自动发送消息。
