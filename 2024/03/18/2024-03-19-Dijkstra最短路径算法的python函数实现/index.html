Dijkstra最短路径算法的Python实现详解

发表于 2024-03-19 更新于 2026-05-14 分类于 数据结构与算法
本文详细讲解了如何使用 Python 实现基于邻接矩阵的 Dijkstra 最短路径算法，包含完整的代码实现、逻辑解析及测试用例。

Dijkstra 算法是经典的单源最短路径算法，用于计算一个节点到其他所有节点的最短路径。本文将详细介绍如何通过 Python 实现该算法，并使用邻接矩阵来存储图结构。

一、 图的邻接矩阵表示

在实现算法之前，我们需要先定义图的数据结构。对于稠密图，邻接矩阵是一个不错的选择。

我们使用一个二维列表 graph 来表示图，其中 graph[u][v] 表示顶点 $u$ 到顶点 $v$ 的边权值：

  • 如果两点之间没有直接连边，则权值记为 $\infty$（无穷大）。
  • 顶点到自身的距离记为 $0$。

二、 Dijkstra 算法核心实现

算法思路

Dijkstra 算法采用贪心策略，主要维护两个集合：

  1. 已确定最短路径的顶点集合 $S$：初始时只包含源点。
  2. 未确定最短路径的顶点集合 $U$：初始时包含除源点外的所有顶点。

算法步骤如下：

  1. 初始化 distance 数组，源点距离为 0，其余为 $\infty$。
  2. 从集合 $U$ 中找出距离源点最近的顶点 $v$，将其加入集合 $S$。
  3. 对顶点 $v$ 的所有邻接点进行“松弛操作”：若经过 $v$ 到达邻接点 $u$ 的距离更短，则更新 distance[u]。
  4. 重复步骤 2 和 3，直到所有顶点都加入集合 $S$。

Python 代码实现

```python
def dijkstra(graph, starting_node, node_num):
“””
Dijkstra最短路径算法实现

Args:
    graph: 图的邻接矩阵
    starting_node: 起始节点索引
    node_num: 节点总数量

Returns:
    list: 包含从起始节点到所有其他节点最短距离的列表
"""

# used 数组：标记顶点是否已访问（是否已加入集合 S）
# used[v] 为 False 说明该顶点还在集合 U 中，否则在集合 S 中
used = [False for _ in range(node_num)]

# distance 数组：记录源点到各个顶点的当前最短距离
# 初始时，源点到自身的距离为0，到其他顶点距离为无穷大
distance = [float('inf') for _ in range(node_num)]
distance[starting_node] = 0

while True:
    # 寻找未访问顶点中距离源点最近的顶点 v
    v = -1
    for u in range(node_num):
        if not used[u] and (v == -1 or distance[u] < distance[v]):
            v = u

    # 如果 v == -1，说明所有顶点都已处理完毕，退出循环
    if v == -1:
        break

    # 将选定的顶点 v 标记为已访问（加入集合 S）
    used[v] = True

    # 松弛操作：遍历所有顶点，更新通过 v 能到达的其他顶点的距离
    for u in range(node_num):
        # 如果通过 v 到达 u 的距离比当前记录的距离更短，则更新
        distance[u] = min(distance[u], distance[v] + graph[v][u])

return distance

时间复杂度分析：
在该实现中，使用了两层循环来查找最小距离节点和更新邻接点，时间复杂度为 $O(V^2)$，其中 $V$ 是顶点数量。这种实现方式适合稠密图。对于稀疏图，通常会结合优先队列（最小堆）进行优化，将复杂度降低至 $O(E \log V)$。
