题目 11：字符串逆序输出

题目分析：
本题要求将输入的字符串反向输出。核心思路是利用循环，从字符串的最后一个字符（下标为 size()-1）开始，向前遍历至第一个字符（下标为 0）。

代码实现：

```cpp

include

include

using namespace std;

int main() {
string a;
cin >> a;

// 从字符串末尾下标开始向前遍历
// 注意：size() 返回无符号数，建议显式转换为 int 或避免 i 为无符号时的下溢问题
for (int i = static_cast<int>(a.size()) - 1; i >= 0; i--) {
    cout << a[i];
}

cout << endl;
return 0;

}
注意点：
在循环条件中，a.size() 返回 size_t（无符号整型）。如果字符串为空，a.size() - 1 不会变成 -1，而是变成一个巨大的正数，导致越界访问。因此，代码中将其强制转换为 int 是一种良好的防御性编程习惯。

题目 13：函数重载与排序

题目分析：
本题演示了函数重载 的概念。针对 int、float、double 三种不同类型的数据，编写同名函数 sortData 进行排序。此外，代码中注释部分展示了如何使用函数模板 来更简洁地实现相同功能。

代码实现：

```cpp

include

include

include

using namespace std;

// 重载函数：用于整数排序
void sortData(vector& data) {
sort(data.begin(), data.end());
}

// 重载函数：用于单精度浮点数排序
void sortData(vector& data) {
sort(data.begin(), data.end());
}

// 重载函数：用于双精度浮点数排序
void sortData(vector& data) {
sort(data.begin(), data.end());
}

/
// 推荐写法：使用函数模板替代重载，减少代码冗余
template
void sortDataTemplate(vector& data) {
sort(data.begin(), data.end());
} /

int main() {
// 初始化测试数据
vector intData = {5, 3, 1, 4, 2};
vector floatData = {5.5f, 3.3f, 1.1f, 4.4f, 2.2f};
vector doubleData = {5.55, 3.33, 1.11, 4.44, 2.22};

// 调用对应的重载函数
sortData(intData);
sortData(floatData);
sortData(doubleData);

// 输出排序结果
cout << "Sorted integers: ";
for (int i : intData) {
    cout << i << " ";
}
cout << endl;

cout << "Sorted floats: ";
for (float f : floatData) {
    cout << f << " ";
}
cout << endl;

cout << "Sorted doubles: ";
for (double d : doubleData) {
    cout << d << " ";
}
cout << endl;

return 0;

}
核心知识点：

  1. 函数重载：允许定义同名函数，但参数类型或数量必须不同。编译器根据调用时的实参类型匹配具体函数。
  2. 函数模板：如代码注释所示，使用模板可以处理任意支持比较操作的类型，是更通用、更现代的 C++ 编程方式。
