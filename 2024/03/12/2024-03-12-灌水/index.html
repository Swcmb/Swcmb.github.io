蓝桥杯真题解析：01矩阵的扩散问题

发表于 2024-03-12 更新于 2026-05-14 分类于 算法题解

题目背景

这是一道经典的搜索与模拟问题，常见于蓝桥杯等算法竞赛中。题目考察了对矩阵数据的处理能力以及对BFS（广度优先搜索）或DFS（深度优先搜索）算法的理解与应用。

问题描述

小蓝有一个 $30$ 行 $40$ 列的 $01$ 矩阵。他打算将第一行第一列的 $0$ 变为 $2$。

变化规则如下：

  1. 变化具有传染性，每次 $2$ 的上下左右四个相邻位置中的 $0$ 都会变成 $2$。
  2. 该过程持续进行，直到最后每个 $2$ 的周围都是 $1$ 或 $2$ 时结束。

请问，最终矩阵中有多少个 $2$？

思路分析

本题本质上是一个 Flood Fill（漫水填充） 问题。

  1. 数据结构选择：
    我们需要处理二维矩阵，首先将字符串数据转换为二维列表（矩阵）。

  2. 算法选择：
    由于扩散是从中心向四周扩散，且每一轮扩散是同时发生的，这非常适合使用 BFS（广度优先搜索）。

    • 初始化一个队列，将起点坐标 (0, 0) 加入队列。
    • 只要队列不为空，就取出队首元素，检查其上下左右四个方向的邻居。
    • 如果邻居是 0，则将其变为 2，并将该邻居坐标加入队列，等待下一轮处理。
    • 如果邻居是 1 或已经处理过（变为 2），则跳过。

  3. 终止条件：
    当队列为空时，说明所有能扩散到的 0 都已经变成了 2，此时算法结束。

1. 数据预处理

去除首尾空白，按行分割，并将每行字符串转换为整数列表

matrix = []
for line in data_str.strip().split(‘\n’):
row = [int(x) for x in list(line.strip())]
matrix.append(row)

rows = len(matrix)
cols = len(matrix[0])

2. BFS 初始化

如果起点本身就是 1，则无法扩散，但根据题目描述，起点是 0 变为 2

if matrix[0][0] == 0:
matrix[0][0] = 2

queue = deque()
queue.append((0, 0))

定义四个方向的偏移量：上、下、左、右

directions = [(-1, 0), (1, 0), (0, -1), (0, 1)]

3. 开始 BFS 扩散

while queue:
x, y = queue.popleft()

for dx, dy in directions:
    nx, ny = x + dx, y + dy

    # 边界检查
    if 0 <= nx < rows and 0 <= ny < cols:
        # 如果相邻位置是 0，则变为 2 并加入队列
        if matrix[nx][ny] == 0:
            matrix[nx][ny] = 2
            queue.append((nx, ny))

4. 统计结果

count = 0
for row in matrix:
for val in row:
if val == 2:
count += 1

print(f”最终矩阵中 2 的个数为: {count}”)

辅助函数：打印矩阵（可选，用于调试）

def print_matrix(m):
for row in m:
print(“”.join(map(str, row)))

print_matrix(matrix)

总结

此类“扩散”、“感染”类问题，核心考点在于：

  1. 正确解析输入数据。
  2. 使用 BFS 模拟扩散过程（先进先出）。
  3. 注意边界条件的判断，防止数组越界。

使用 Python 的 collections.deque 作为队列，可以保证算法的高效执行，时间复杂度为 $O(N \times M)$。
