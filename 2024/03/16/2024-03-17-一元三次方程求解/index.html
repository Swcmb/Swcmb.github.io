一元三次方程求解

发表于 2024-03-17 更新于 2026-05-14 分类于 算法笔记

题目描述

给定一个形如 $ax^3 + bx^2 + cx + d = 0$ 的一元三次方程，已知各项系数（$a, b, c, d$ 均为实数）。约定该方程存在三个不同实根（根的范围在 -100 至 100 之间），且根与根之差的绝对值 $\ge 1$。

要求：由小到大依次在同一行输出这三个实根，根与根之间留有空格，结果精确到小数点后 2 位。

解题思路

本题是一道经典的数值分析问题，主要考察二分法求解方程根的应用。由于题目已经给出了明确的根的存在区间以及根之间的距离性质，我们可以利用单调性和零点存在定理来求解。

关键性质分析

  1. 连续性与零点定理：
    对于多项式函数 $f(x) = ax^3 + bx^2 + cx + d$，它在实数域上是连续的。如果在区间 $[left, right]$ 上满足 $f(left) \cdot f(right) < 0$，则该区间内必定存在一个实根。

  2. 根的分布特征：
    题目指出“根与根之差的绝对值 $\ge 1$”。这意味着在长度为 1 的相邻整数区间内（例如 $[0, 1]$、$[1, 2]$ 等），最多只可能存在一个实根。这为我们划分求解区间提供了依据。

算法流程

  1. 遍历区间：
    由于根的范围在 $[-100, 100]$ 之间，我们可以从 $-100$ 遍历到 $100$，每次检查长度为 1 的子区间 $[i, i+1]$。

  2. 判断根的存在：

    • 若 $f(i) = 0$，说明左端点恰好是一个根（根据题目条件，区间内不会同时存在两个根，且根不重合）。
    • 若 $f(i) \cdot f(i+1) < 0$，根据零点定理，说明根在区间 $(i, i+1)$ 内部。

  3. 二分逼近：
    对于存在根的区间 $(left, right)$，使用二分法不断缩小范围：

    • 取中点 $mid = (left + right) / 2$。
    • 判断 $f(mid)$ 的符号，确定根是在左半部分还是右半部分，更新区间端点。
    • 根据题目精度要求（保留 2 位小数），我们可以设定循环终止条件为区间长度 $< 0.001$，或者直接循环固定次数（如 50-60 次）以保证精度。

def f(x):

return a (x ** 3) + b (x * 2) + c x + d

for i in range(-100, 100):
left = i
right = i + 1

y1 = f(left)
y2 = f(right)

# 情况1：左端点恰好是根
# 题目保证根之间差值>=1，故若左端点是根，右端点不可能是根，直接输出
if y1 == 0:
    print("{:.2f}".format(left), end=' ')

# 情况2：根在区间内部 (异号)
if y1 * y2 < 0:
    # 二分法求解
    # 精度控制：由于保留2位小数，循环至区间足够小即可
    while right - left >= 0.001:
        mid = (left + right) / 2
        if f(mid) * f(right) <= 0:
            # 根在 [mid, right] 区间
            left = mid
        else:
            # 根在 [left, mid] 区间
            right = mid

    # 输出右端点（此时左右端点已非常接近）
    print("{:.2f}".format(right), end=' ')

关键点总结

  1. 输入类型：题目明确系数为实数，因此 input 读取后应转换为 float 而非 int，这是容易丢分的细节。
  2. 端点处理：在遍历过程中，如果直接判断 f(i) == 0 输出，需要注意区间端点的重合问题。但由于题目保证根差值 $\ge 1$，且区间步长为 1，直接输出左端点不会导致重复输出（因为下一个区间的左端点就是当前区间的右端点，而右端点如果也是根，会在下一个循环作为左端点处理，或者通过异号条件处理）。
  3. 精度控制：二分法的循环条件通常设置为区间长度小于一个很小的阈值（如 $10^{-3}$ 或 $10^{-4}$），足以满足题目“保留两位小数”的要求。
