蓝桥杯专题：质因数分解

发表于 2024-03-13 更新于 2026-05-14 分类于 算法刷题 ， 蓝桥杯
本文详细解析蓝桥杯基础题“质因数分解”，利用质数的数学性质优化解题思路，提供Python代码实现及复杂度分析。

1. 题目描述

题目链接：蓝桥杯题库 - 质因数分解

已知正整数 $n$ 是两个不同的质数的乘积，试求出较大的那个质数。

输入格式

输入只有一行，包含一个正整数 $n$。

数据范围：$6 < n < 2^{10}$

输出格式

输出只有一行，包含一个正整数 $p$，即较大的那个质数。

2. 题目分析与思路

核心思路

题目明确指出：$n$ 是两个不同质数的乘积。这是一条非常关键的信息。

设这两个质数分别为 $p_1$ 和 $p_2$（不妨设 $p_1 < p_2$），则有：

根据算术基本定理的性质，如果我们在 $[2, n-1]$ 的范围内寻找 $n$ 的因数，找到的第一个能整除 $n$ 的数，必然是最小的那个质因数 $p_1$。

为什么？

  1. 因为 $p_1$ 是质数，且 $p_1 < p_2$，所以 $p_1$ 是 $n$ 除了 1 以外最小的因数。
  2. 一旦找到 $p_1$，根据乘法关系，较大的那个质数 $p_2$ 自然就是 $n / p_1$。

算法优化

虽然题目数据范围较小（$n < 1024$），直接从 2 遍历到 $n$ 完全可以通过测试，但作为技术博主，我们需要考虑到时间复杂度。

实际上，寻找因数只需要遍历到 $\sqrt{n}$ 即可。如果 $n$ 有因数 $i$，那么 $n/i$ 也是 $n$ 的因数。两者中必有一个小于等于 $\sqrt{n}$。因此，遍历上限设为 $\sqrt{n}$ 可以将时间复杂度从 $O(n)$ 降低到 $O(\sqrt{n})$。

核心逻辑：任何一个大于1的整数都可以分解成若干个质数的连乘积

题目特例：n是两个不同质数的乘积，故找到的最小因数即为较小质数

import math

n = int(input())

方法一：遍历到 n (本题数据量小，足以通过)

for i in range(2, n):

if n % i == 0:

print(n // i)

break

方法二：优化遍历至 sqrt(n) (更优解法)

for i in range(2, int(math.sqrt(n)) + 1):
if n % i == 0:

    # i 是较小的质因数，n // i 即为所求的较大质因数
    print(n // i)
    break

4. 复杂度分析

  • 时间复杂度：$O(\sqrt{n})$。由于我们只需要找到第一个因数，最坏情况下遍历到 $\sqrt{n}$ 即可找到。对于题目给定的 $n < 1024$，循环次数极少，效率极高。
  • 空间复杂度：$O(1)$。只使用了常数级变量。

5. 总结

这道题虽然基础，但考查了对数学性质的应用：

  1. 如果能利用“不同质数乘积”这一约束条件，可以避免复杂的质数判断逻辑。
  2. 找到最小因数，即可推导出最大因数，这是解题的关键跳跃。

注意：在处理因数相关问题时，遍历上界通常设为 $\sqrt{n}$，这是一个非常重要的编程技巧。
