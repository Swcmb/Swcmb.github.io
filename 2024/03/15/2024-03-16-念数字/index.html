L1-007 念数字

发表于 2024-03-16 更新于 2026-05-14 分类于 算法刷题

题目描述

题目编号： L1-007
题目来源： 团体程序设计天梯赛-练习集

输入一个整数，输出每个数字对应的拼音。当整数为负数时，先输出 fu 字。十个数字对应的拼音如下：

数字 拼音
0 ling
1 yi
2 er
3 san
4 si
5 wu
6 liu
7 qi
8 ba
9 jiu

输入格式

输入在一行中给出一个整数，例如：1234。

注意： 整数包括负数、零和正数。

输出格式

在一行中输出这个整数对应的拼音，每个数字的拼音之间用空格分开，行末没有最后的空格。

输入输出样例

输入样例 1：
-600
输出样例 1：

fu liu ling ling

题目解析

本题属于基础的字符串处理问题。主要考察点包括：

  1. 字符串与数值的处理：将输入视为字符串处理更为便捷，可以避免数值溢出问题，同时也方便逐位转换。
  2. 映射关系的建立：需要建立数字字符（’0’-‘9’）与拼音字符串之间的对应关系。可以使用字典或列表实现。
  3. 特殊情况处理：需要注意负数符号 - 的单独处理，将其映射为 fu。

算法思路

  1. 构建数字到拼音的映射表。
  2. 读取输入的整数字符串。
  3. 判断字符串首字符是否为 -：
    • 若是，先输出 fu，然后处理剩余字符。
    • 若否，直接处理所有字符。
  4. 遍历数字字符，查找映射表获取对应拼音，拼接结果。
  5. 控制输出格式，确保末尾无多余空格。

代码实现

以下是优化后的 Python 代码实现。相比原始代码，使用了字典结构存储映射关系，逻辑更加清晰，代码可读性更强。

```python
def read_number():

# 定义数字到拼音的映射字典
pinyin_map = {
    '0': 'ling', '1': 'yi', '2': 'er', '3': 'san', '4': 'si',
    '5': 'wu', '6': 'liu', '7': 'qi', '8': 'ba', '9': 'jiu'
}

# 读取输入
num_str = input().strip()
result = []

# 处理负数情况
if num_str.startswith('-'):
    result.append('fu')
    num_str = num_str[1:] # 截取数字部分

# 遍历剩余字符进行转换
for char in num_str:
    # 字典查找，直接获取拼音
    if char in pinyin_map:
        result.append(pinyin_map[char])

# 输出结果，用空格连接
print(' '.join(result))

if name == ‘main‘:
read_number()

代码优化点说明

  1. 数据结构优化：原代码使用了元组列表 li 并通过索引 int(i) 查找，逻辑较为绕弯。优化后直接使用字典 pinyin_map，通过 Key-Value 键值对直接映射，时间复杂度为 O(1)，且代码更直观。
  2. 逻辑简化：原代码在循环中重复判断了 if (i == li[int(i)][0])，这步判断是多余的，因为索引本身就是为了查找对应项。优化后直接遍历字符串字符并映射。
  3. 可读性：使用 startswith 判断负号，语义明确；使用列表 append 方法收集结果，最后统一 join 输出，符合 Pythonic 风格。
