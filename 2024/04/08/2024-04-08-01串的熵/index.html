蓝桥杯专题：01串的熵问题解析

发表于 2024-04-08 分类于 算法笔记
本文详细解析了蓝桥杯‘01串的熵’问题，通过Python代码实现信息熵公式的逆向求解，定位满足条件的0出现次数。

题目背景

本题选自蓝桥云课/蓝桥杯真题，主要考察信息论中熵（Entropy）概念的应用。

对于一个长度为 $L$ 的01串，若其熵值已知，我们需要反推其中 0 或 1 的出现次数。题目给出了具体的熵值目标，要求通过编程进行求解。

思路分析

  1. 确定变量：

    • 设定总长度 $x = 23333333$。
    • 设 0 出现的次数为 $i$，则 1 出现的次数为 $j = x - i$。
    • 为了优化计算效率，我们只需遍历 $i$ 从 $1$ 到 $x//2$ 即可。因为当 0 的数量小于一半时，1 的数量必然大于一半，覆盖了对称的情况。

  2. 计算公式：

    • 根据题目给出的特定熵值计算方式（注意：题目定义的熵计算可能包含特定的加权或缩放，代码需严格遵循题目给出的数值匹配逻辑）。
    • 计算每一项的值：$(j/x) \times \log_2(j/x) \times j$（注意符号处理）。
    • 累加各项结果，判断是否等于目标值 -11625907.5798。

  3. 精度处理：

    • 由于浮点数计算存在精度误差，使用 round() 函数保留4位小数进行比对。

代码实现

以下是优化后的 Python 解题代码：

```python
import math

def calculate_entropy_zeros():

# 定义总长度
total_length = 23333333

# 目标熵值
target_entropy = -11625907.5798

# 遍历0出现的次数 i
# 范围设定为 1 到 total_length//2，利用对称性减少计算量
for i in range(1, total_length // 2):
    j = total_length - i  # 1出现的次数

    # 计算当前分布下的熵值项
    # 公式推导：- [ (j/x)*log2(j/x)*j + (i/x)*log2(i/x)*i ]
    # 注意：代码中直接计算了负号后的项，目标值本身为负，故公式内部为正
    p_j = j / total_length
    p_i = i / total_length

    val_j = p_j * math.log2(p_j) * j
    val_i = p_i * math.log2(p_i) * i

    current_val = val_j + val_i

    # 四舍五入保留4位小数进行比较
    if round(current_val, 4) == target_entropy:
        print(f"找到匹配结果：0出现的次数为 {i}")
        print(f"1出现的次数为 {j}")
        return i

总结

在处理涉及浮点数精度匹配的算法题时，关键在于：

  1. 正确理解题目定义的数学公式。
  2. 利用对称性或单调性优化遍历范围。
  3. 使用 round 函数处理浮点数计算误差。
