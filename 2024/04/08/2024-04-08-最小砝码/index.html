蓝桥杯真题：最小砝码（贪心与数学推导)

发表于 2024-04-08 更新于 2026-05-14 分类于 Algorithm

问题描述

你有一架天平。现在你要设计一套砝码，使得利用这些砝码可以称出任意小于等于 $N$ 的正整数重量。那么这套砝码最少需要包含多少个砝码？注意砝码可以放在天平两边。

输入格式

输入包含一个正整数 $N$。

输出格式

输出一个整数代表答案。

题目分析与推导

这是一道经典的数学推导与贪心算法结合的题目。我们需要找到能覆盖 $[1, N]$ 范围内所有整数重量的最少砝码组合。

1. 贪心策略的选择

砝码可以放在天平的两端，这意味着对于每一个砝码，它有三种状态：

  1. 放在左盘（记为 $+w$）。
  2. 放在右盘（记为 $-w$）。
  3. 不放（记为 $0$）。

这实际上对应了平衡三进制（Balanced Ternary）的概念。

2. 为什么选择 $1, 3, 9, \dots$？

假设我们已经有一组砝码，能称出的最大重量为 $M$。当我们加入一个新的砝码 $w$ 时，由于砝码可以放在两边，我们能称出的新范围会发生变化。

设当前砝码能称出的最大重量为 $K$（即能覆盖 $[1, K]$ 的所有整数）。
当我们加入一个新砝码 $w$ 时，新的可称量范围变为：

  • 原有范围：$[1, K]$
  • 新增范围（利用新砝码 $w$）：
    • $w$ 单独：$w$
    • $w + [1, K]$：$[w+1, w+K]$
    • $w - [1, K]$（放在异侧）：$[w-K, w-1]$

3. 推导过程

根据上述推导，我们来确定砝码序列：

  1. 第一个砝码：初始 $K=0$，则 $w_1 = 2(0) + 1 = 1$。
    • 当前最大称重 $K_{new} = 1$。
  2. 第二个砝码：当前 $K=1$，则 $w_2 = 2(1) + 1 = 3$。
    • 当前最大称重 $K_{new} = 1 + 3 = 4$（覆盖 $1$ 到 $4$）。
  3. 第三个砝码：当前 $K=4$，则 $w_3 = 2(4) + 1 = 9$。
    • 当前最大称重 $K_{new} = 4 + 9 = 13$（覆盖 $1$ 到 $13$）。
  4. 第四个砝码：当前 $K=13$，则 $w_4 = 2(13) + 1 = 27$。
    • 当前最大称重 $K_{new} = 13 + 27 = 40$。

结论：最优的砝码组合是 $3$ 的幂次方（$1, 3, 9, 27, \dots$）。
