urllib实战：Ajax GET请求抓取豆瓣电影数据

发表于 2024-07-26 更新于 2026-05-14 分类于 Python爬虫
本文详细讲解如何使用Python标准库urllib处理Ajax GET请求，通过模拟浏览器Header抓取豆瓣电影排行榜数据，并进行本地JSON存储。

前言

在网络爬虫开发中，很多网站（如豆瓣）采用前后端分离的架构，数据通过 Ajax 异步加载。这意味着直接请求页面 URL 无法获取到动态渲染的电影数据，我们需要分析 Ajax 接口，直接请求 API。

本文将以豆瓣电影排行榜为例，演示如何使用 Python 标准库 urllib 发送带参数的 GET 请求，并保存返回的 JSON 数据。

目标分析

本次抓取的目标是豆瓣电影排行榜。

接口地址：
https://movie.douban.com/j/chart/top_list

关键参数：

  • type: 电影类型（如 17 代表科幻片）
  • interval_id: 区间段（如 100:90 代表评分区间）
  • start: 起始索引（分页参数，从 0 开始）
  • limit: 每页显示数量（如 20）
