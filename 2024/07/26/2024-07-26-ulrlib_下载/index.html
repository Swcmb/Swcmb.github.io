Python爬虫基础：使用urllib下载网页、图片与视频

发表于 2024-07-26 更新于 2026-05-14 分类于 Python爬虫
本文详细介绍如何使用Python标准库urllib.request模块中的urlretrieve方法，实现网页源码、图片及视频文件的本地下载，并附带代码示例与参数解析。

在Python网络爬虫开发中，将网络资源下载到本地是常见的需求。虽然可以使用 requests 库通过流式写入实现，但 Python 标准库 urllib 提供的 urlretrieve 方法同样简洁高效，无需安装第三方库即可完成下载任务。

本文将演示如何利用 urllib.request.urlretrieve 快速下载网页、图片和视频文件。

1. 核心方法介绍

urllib.request.urlretrieve 方法用于将远程数据直接下载到本地文件。其常用参数如下：

  • url：需要下载的资源链接。
  • filename：下载后保存的本地文件名（包含路径）。
  • reporthook（可选）：回调函数，用于显示下载进度条。
  • data（可选）：POST请求的数据。

2. 代码实现

以下代码演示了三种不同类型资源的下载方式：网页HTML、图片和视频文件。

```python
import urllib.request

==========================================

1. 下载网页内容

==========================================

指定目标网页URL

url_page = ‘http://www.baidu.com‘

urlretrieve(url, filename)

url: 下载路径

filename: 保存的文件名

urllib.request.urlretrieve(url_page, filename=”baidu.html”)
print(“网页下载完成：baidu.html”)

==========================================

2. 下载图片资源

==========================================

url_image = ‘https://gips1.baidu.com/it/u=869589177,1848598878&fm=3028&app=3028&size=w931&q=75&n=0&f=PNG&fmt=auto&maxorilen2heic=2000000‘

将图片保存为 image.jpg

urllib.request.urlretrieve(url=url_image, filename=’image.jpg’)
print(“图片下载完成：image.jpg”)

==========================================

3. 下载视频资源

==========================================

url_video = ‘https://vdept3.bdstatic.com/mda-qgd2891c16cc1p35/360p/h264/1720920969149508765/mda-qgd2891c16cc1p35.mp4?v_from_s=hkapp-haokan-nanjing&auth_key=1720950217-0-0-c76b0b6fef675d36234a30b6db923966&bcevod_channel=searchbox_feed&pd=1&cr=0&cd=0&pt=3&logid=2617830577&vid=15798241631673631806&klogid=2617830577&abtest=101830_2-17451_1-3000225_1‘

将视频保存为 video.mp4

urllib.request.urlretrieve(url_video, filename=”video.mp4”)
print(“视频下载完成：video.mp4”)

3. 注意事项

  1. 文件扩展名：在使用 urlretrieve 时，建议手动指定正确的文件扩展名（如 .html, .jpg, .mp4），该方法本身不会自动根据 Content-Type 识别格式。
  2. 大文件下载：对于大文件（如视频），urlretrieve 会阻塞主线程直到下载完成。如果需要显示下载进度，可以利用第三个参数 reporthook 编写回调函数。
  3. 链接时效性：示例中的图片和视频链接可能包含时效性参数（如 auth_key），如果运行代码报错 HTTP Error 403 或 404，请替换为有效的URL进行测试。

4. 总结

urllib.request.urlretrieve 是一个轻量级的文件下载解决方案，特别适用于编写简单的爬虫脚本或批量下载工具。掌握其基本用法是 Python 爬虫入门的重要一步。
