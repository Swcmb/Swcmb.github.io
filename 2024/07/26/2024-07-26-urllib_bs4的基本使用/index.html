Python爬虫基础：urllib与BeautifulSoup4实战指南

发表于 2024-07-26 更新于 2026-05-14 分类于 爬虫技术
本文详细讲解Python内置库urllib与第三方库BeautifulSoup4的基础用法，通过实战代码演示如何发送请求、解析HTML文档及提取数据，适合爬虫初学者入门。

前言

在Python爬虫开发中，urllib 和 BeautifulSoup4 (简称 bs4) 是一对经典的黄金搭档。urllib 是 Python 内置的标准库，无需安装即可使用，主要负责发送网络请求；而 BeautifulSoup4 则是强大的第三方解析库，用于将复杂的 HTML 文档解析成树形结构，便于数据提取。

本文将从零开始，演示这两个库的基本使用流程。

一、环境准备

urllib 是 Python 内置库，直接导入即可使用。但 BeautifulSoup4 需要额外安装。请确保你的 Python 环境已配置好，并在终端执行以下命令安装解析库：

``bash pip install beautifulsoup4 *建议同时安装lxml解析器以获得更快的解析速度：pip install lxml`*

二、urllib：发送网络请求

urllib 库中最核心的模块是 request，它模拟了浏览器向服务器发送请求的过程。

1. 基本GET请求

下面演示如何使用 urllib.request 获取百度首页的 HTML 源码：

```python

导入请求模块

import urllib.request

1. 定义目标URL

url = ‘https://www.baidu.com‘

2. 发送请求，获取响应对象

response = urllib.request.urlopen(url)

3. 读取并解码响应内容

注意：网页内容是bytes类型，需要decode解码为str

html_content = response.read().decode(‘utf-8’)

4. 打印结果

print(html_content)

2. 伪装请求头 (User-Agent)

在实际爬虫过程中，直接使用 urlopen 会被很多网站识别为爬虫并拒绝访问。此时我们需要伪装成浏览器，这需要用到 urllib.request.Request 对象来封装请求头。

```python
import urllib.request

url = ‘https://www.baidu.com‘

模拟浏览器的User-Agent

headers = {
‘User-Agent’: ‘Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/126.0.0.0 Safari/537.36’
}

封装Request对象

request = urllib.request.Request(url=url, headers=headers)

发送封装后的请求

response = urllib.request.urlopen(request)
content = response.read().decode(‘utf-8’)

print(content)

三、BeautifulSoup4：解析HTML文档

拿到网页源码后，我们需要从中提取有用的数据。BeautifulSoup 可以将复杂的 HTML 文档转换成一个树形结构，每个节点都是 Python 对象。

1. 初始化对象

我们需要将 urllib 获取到的字符串源码传递给 BeautifulSoup。

```python
from bs4 import BeautifulSoup

假设 file 是上文获取到的HTML字符串

使用 ‘html.parser’ 或 ‘lxml’ 作为解析器

soup = BeautifulSoup(content, ‘html.parser’)

美化输出（格式化HTML代码，便于调试）

print(soup.prettify())

2. 节点定位方法

BeautifulSoup 提供了多种定位方法，最常用的是 find 和 find_all。

(1) 根据标签名查找

```python

查找第一个a标签

first_a = soup.a
print(first_a)

查找所有a标签

all_a = soup.find_all(‘a’)
for a in all_a:
print(a)

(2) 根据属性查找

在实际开发中，我们更多是配合 class 或 id 属性进行精确定位。

```python

查找 class_=’s-top-left’ 的 div 标签

注意：class是Python关键字，BeautifulSoup中使用 class_

divdata = soup.find(‘div’, class=’s-top-left’)
print(div_data)

查找特定属性的标签

data = soup.find(‘div’, attrs={‘id’: ‘header’})

3. 提取数据

定位到标签后，最重要的步骤是提取其中的文本或链接地址。

```python

假设我们已经定位到了某个 a 标签

item = soup.find(‘a’)

获取标签之间的文本内容

text = item.text
print(f”文本内容: {text}”)

获取标签属性的值 (如 href 链接)

href = item.get(‘href’)
print(f”链接地址: {href}”)

四、综合实战案例

下面将 urllib 和 bs4 结合，演示一个完整的爬虫流程：请求 -> 解析 -> 保存数据。

```python
import urllib.request
from bs4 import BeautifulSoup

def main():

# 1. 确定URL
url = 'https://www.baidu.com'

# 2. 构造请求头
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/126.0.0.0 Safari/537.36'
}

# 3. 发送请求
request = urllib.request.Request(url=url, headers=headers)
response = urllib.request.urlopen(request)
html_data = response.read().decode('utf-8')

# 4. 实例化BeautifulSoup对象
soup = BeautifulSoup(html_data, 'html.parser')

# 5. 提取数据 (例如提取所有链接)
links = soup.find_all('a')

# 6. 数据处理与输出
print(f"共找到 {len(links)} 个链接:")
for link in links:
    href = link.get('href')
    text = link.text.strip()
    if href:
        print(f"文本: {text} -> 链接: {href}")

if name == ‘main‘:
main()

五、总结

urllib 与 BeautifulSoup4 的组合非常适合小型爬虫项目或学习爬虫原理。

  • urllib：负责网络连接，重点在于处理 URL 编码、请求头伪装和 Cookie 处理。
  • BeautifulSoup：负责数据清洗，重点在于熟练使用选择器 (find, select) 提取目标数据。

掌握这两个库，是迈向 Python 爬虫高手的第一步。
