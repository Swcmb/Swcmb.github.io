Python 爬虫基础：urllib 模块快速上手

发表于 2024-07-26 更新于 2026-05-14 分类于 Python爬虫

前言

在 Python 爬虫开发中，urllib 是 Python 内置的标准库，无需额外安装即可使用。它提供了丰富的接口用于处理 URL 和网络请求。本文将通过一个简单的示例——获取百度首页源码，带你快速掌握 urllib 的基本使用流程。

完整代码示例

下面是一段最基础的爬虫代码，实现了向百度发送请求并获取页面 HTML 源码的功能：

```python

1. 导入 urllib 库的 request 模块

import urllib.request

2. 定义目标 URL (即你要访问的地址)

url = ‘http://www.baidu.com‘

3. 模拟浏览器向服务器发送请求

response 对象包含服务器返回的所有信息

response = urllib.request.urlopen(url)

4. 获取响应中的页面源码

read() 方法返回的是字节形式的二进制数据

需要使用 decode(‘utf-8’) 将二进制数据转换为字符串

content = response.read().decode(‘utf-8’)

5. 打印获取到的内容

print(content)

代码核心逻辑解析

1. 发送请求

``python response = urllib.request.urlopen(url)urlopen是urllib.request中最核心的方法，它可以像浏览器一样向指定的 URL 发送 GET 请求。返回的response` 对象包含了服务器的响应状态、响应头以及响应体。

2. 读取与解码

```python
content = response.read().decode(‘utf-8’)
这一步包含两个关键操作：

  • read(): 读取响应体的内容。值得注意的是，该方法返回的是二进制数据，通常格式为 <class 'bytes'>。直接打印会显示为 b'...' 的形式，可读性差。
  • decode('utf-8'): 为了将二进制数据还原成人类可读的字符串，我们需要进行解码。因为百度首页的编码格式通常为 UTF-8，所以这里使用 UTF-8 进行解码。

知识点总结

步骤 方法/变量 说明
导入模块 import urllib.request 引入处理请求的标准库。
定义地址 url 目标网站的统一资源定位符。
发起请求 urllib.request.urlopen(url) 向服务器发起 HTTP 请求。
读取数据 response.read() 获取二进制格式的网页源码。
解码数据 .decode('utf-8') 将二进制字节流转换为字符串。

结语

通过以上步骤，我们成功实现了获取网页源码的第一步。虽然 urllib 功能强大，但在实际开发中，由于其 API 相对底层，许多开发者也会选择使用更加便捷的第三方库如 requests。但理解 urllib 的原理对于掌握 Python 爬虫至关重要。
